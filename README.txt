--------------------------------------
|Proyecto No.1 - Bioshock page clone|
--------------------------------------

Description:
Using a webpack boiler plate, this project 
is a copy of the Bioshock infinite official 
site.

Installation:
    npm install

Usage:
Go to the root directory, and run:
    
    npm run build

Usage:
And webpack will generate in the dist folder
the resources necessary to run the page. Then 
just mount the page on a server, or double click
it, to run on your default browser
